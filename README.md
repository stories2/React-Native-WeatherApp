# React-Native-WeatherApp
리엑트 네이티브로 만들어보는 날씨 앱

## Develop

### Run

```
cd ./fucking-weather
yarn start
```

And capture QR code using mobile phone.

- 상황에 따라서는 `LAN` 옵션 대신 `TUNNEL`을 사용해야 빌드가 되는 경우가 있음
